Пользовательское соглашение

  1. Общие положения
    1. USSR games (далее - Игры) предлагает пользователю сети Интернет (далее – Пользователь) - использовать собственно разработанную версию игры змейки, изложенных в настоящем Пользовательском соглашении (далее — «Соглашение», «ПС»).

  2. Сведения об использовании игры на ПК
    1. Упраление игрой возможно как клавиатурой так и клавишами.
    2. В версии для пк есть некотрые полезные сочетания клавиш:
      • Клавиша Щ (O) обнуляет значение счетчика;
      • Клавиша Ж (;) позволяет писать текст под счетчиком;

  3. Сведения об использовании игры на Android
    1. Упраление игрой возможно только клавишами.
    2. В этой версии отсутствуют некоторые функции, которые есть в компьютерной версиию. Но вскором времени они будут добавлены.

  4. Сведения об использовании игры на Apple
    1. Пока что версии для apple нет, но она разрабатывается. А пока вы можете поиграть змейку, предназначенную для Android.
    2. Если у пользователя есть iPad, то пользователь может поиграть змейку, предназначенную для ПК.
    3. Самое главное для пользователей apple - играйте в версию игры для Android с осторожностью, чтобы дядя Тим не забанил вас :)

  5. Реклама
    1. Мы несем ответственность за рекламу, размещенную нами на сайте, в пределах, установленных законодательством РФ.
  6. Исходный код
    1. Исходный код игры является открытым и пользователь может посмотреть его на Github.
